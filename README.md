# React Tutorial

A modern React application built with Vite and TypeScript, organized with a scalable feature-based folder structure.

## Prerequisites

Before you begin, ensure you have the following installed:

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- npm (comes with Node.js)

Verify your installation:

```bash
node --version
npm --version
```

## Installation
```bash
npm create vite@latest
```

## Run dev server
```bash
npm run dev
```
